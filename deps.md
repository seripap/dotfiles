## Install brew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Install other dependencies

```
brew install pure
brew install fzf
brew install bat
brew install the_silver_searcher
brew install ripgrep
brew install awscli
brew install gdal
brew install jq
```
